Micro-interactions (from NN/g research):
• Press feedback on every button — active:scale-[0.97], <100ms (the existing env-sync pattern).
• Selected file uses 3 cues, not color alone: fill + left-edge accent bar + bold (NN/g: differentiate with multiple signals).
• Dirty state shown twice: glowing amber dot in the list + "● unsaved" by the filename + the Save button lighting up.
• Click-to-edit row: focused row gets a blue ring; commit on blur/Enter, Esc cancels (modes must be visible & escapable).
• Masked by default, per-row eye on hover + global "Reveal all" (NN/g: mask + reveal beats either alone).
• Live duplicate-key warning inline next to the field (validation that prevents a real mistake fires live).
• Save → optimistic, toast "Saved" (passive confirmation only); errors stay inline, never in a toast.
• Animations: modal in ~300ms / out ~200ms, ease-out; honor prefers-reduced-motion.
