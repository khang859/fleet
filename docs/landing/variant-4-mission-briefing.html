Copilot for Claude Code

A floating desktop mascot for macOS that watches over all your Claude Code sessions running in Fleet. Approve permissions without switching windows, view live chat history, and send messages directly to Claude.

1200 × 600 — Copilot Overlay with Permission Approval

When Claude Code needs permission to run a command, a panel appears with Allow/Deny buttons — no terminal prompt, no context switch. See every Claude Code session's status at a glance, read full chat history, and type messages to Claude when it's waiting for input. Drag the mascot anywhere on your macOS desktop.
