Copilot for Claude Code

A floating desktop mascot that monitors all your Claude Code sessions. Approve permissions, read chat history, and send messages to Claude — without leaving whatever you're working on.

When Claude needs permission to run a command, Allow/Deny buttons appear instantly — no terminal switching. See every session's status at a glance and type messages directly when Claude is waiting for input.
